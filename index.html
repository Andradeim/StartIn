OBJETIVO

START IN

Plataforma onde os usuários irão reunir e compartilhar suas experiências para o restante das pessoas. O foco da Start In é disseminar o que são as Objetivos de Desenvolvimento Sustentável e, por meio das experiências compartilhadas, mostrar que cada pessoa pode ajudar e mudar o mundo de outras pessoas!

COMO PARTICIPAR

Assim como as outras redes sociais, na Start In, o usuário poderá publicar vídeos, imagens e pequenos textos. Contudo, o foco aqui é tratar do Objetivo de Desenvolvimento Sustentável 2 e de mostrar como qualquer um pode contribuir para sua causa! Ao lado o botão o leverá a um modelo prévio de como será o espaço das publicações.
